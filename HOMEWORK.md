# 作业内容

## Git 小作业

### 任务描述

本次作业需要你在后端实现武将"凌操"的技能。

**技能效果**：在摸牌阶段，凌操的摸牌数量 = 3 + (装备数量 ÷ 2) 向下取整

### 实现要求

- 前端和配置文件中已经实现了对应的功能，你只需要完成后端实现
- 请阅读项目架构文档（`DESIGN.md` 和 `backend/README.md`），理解项目结构，找到合适的后端实现位置
- 需要实现武将技能的核心逻辑

### 验收标准

1. 从你的账号上将仓库 clone 到本地
2. 运行单元测试（`pytest tests/`）
3. 初始状态下会有三个单元测试无法通过
4. 完成作业后，所有单元测试应能通过

### 注意事项

- 请仔细阅读项目架构文档，理解代码结构
- 确保实现的技能逻辑**只在技能发动时生效**
- 完成实现后运行测试验证功能正确性
